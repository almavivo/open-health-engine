## Guidance

**Dose:** Daily standardized extract (typically 300 mg); takes 8–12 weeks.

**Timing:** Evening with food; some users get GI upset on empty stomach.

**Reassessment:** Reassess after 8 to 12 weeks.

## Rationale

- Among the better-evidenced botanicals for memory consolidation in adults.
- Slow-onset; not useful as an acute focus tool.
